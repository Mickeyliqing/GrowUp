# 关于未来

## 第一部分

1.  思考清楚自己从事的行业方向，写下来要搭建的能力模型以及对应的知识体系。
2.  删除所有的互联网课程，每天独立思考并记录，阅读最经典的基础理论书籍，并做读书笔记强化印象。
3.  疯狂学习一切关于效率的知识并练习，成为一个高效的投入最少精力并获得最优收获的人。
4.  回归那个有趣的你，喜欢什么就尽力地去玩去体会。

## 第二部分

1.  重建现实，抛弃一切的道理和教条，靠自己的双眼去从新理解当下这个社会，直面现实，把关于生活的一切幻想都放下。
2.  拿自己做实验，不断拿自己的生活，工作，情感上各种事情来做实验，找到自己最佳的节奏和最佳的模式，在慢慢培养。
3.  意识到这世上大多数人不是骗子就是傻子，媒体，信息，内容，知识的背后都是生意。
4.  近乎残暴地训练你的判断力，筛选能力，和专注能力。
5.  尽最大努力地简化自己的工作和生活。
6.  锻炼无条件的自信和爱自己的能力。
